Functions

1. Intro

A function is a list of instructions (code) that accomplishes a task. A method is a special kind of function that belongs to a type. We've seen examples of both of these already.

  1. print(“Some text”) is an example of calling the print() function which takes a String and prints it to standard output.
  2. contains() is an example of a method which belongs to the String type. In order to call the contains method, we must use an instance of the String type.

2. Function Syntax

In Swift, functions are defined using the following syntax: 

              
  func nameOfFunction() {
      // body of function goes here
  }

3. Functions with Parameters 

              
  func sayHelloToStudent(student: String) {
      print("Hello, \(student)!")
  }
Parameters vs. Arguments

There are some special terms here that require special attention. When we look at the definition of a function like sayHelloToStudent(student: String), any input values that are specified are called parameters. Then, when we actually call a function, as in sayHelloToStudent(student: jarrod) the input values that are passed to the function are known as arguments.

As you may have noticed, inside the body of the function, we access input values using the parameter names. In this case, we just have the single parameter called student. What if there were multiple parameters?
